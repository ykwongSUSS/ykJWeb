Using Franklin

dir: /myJulia

  1. For now, we use VSCode and Franklin to generate this website. There is a great tutorial on youTube: www.youtube.com/watch?v=fVBiLEtZB7A

  2. Once we setup the website (using Franklin theme: jemdoc), we reproduce it from VSCode to Github Desktop

  3. We then link the github Desktop to github.com for synchronization

Required:

  1. julia extension on VSCode alt-j, alt-o

  2. http live-server on VSCode

  3. Franklin addin package to Julia

  4. Operate from Terminal

Starting Julia from Terminal

  1. alt-j, alt-o

  2. pwd() to check current directory

  3. This readme.md file is here: "/Users/ykwong/myJulia/julia-Franklin"

  4. If new: mkdir for website folder ykWeb

  5. cd("ykWeb")

  6. julia> ]

  7. pkg> add Franklin

  8. pkg> status

Create and serve newsite

  1. julia> using Franklin

  2. newsite("ykJWeb"; template="jemdoc")

  3. current directory is ykJWeb

  4. pkg> activate . #impt: to activate at this directory

  5. julia> serve() # then can use serve() if stopped

Serving from another web

  1. cd("../SandBox")

  2. pwd() to make sure in the sandbox web directory

  3. pkg> activate . #impt: to activate at this directory
